Capabilities

The Spec Sheet

Everything you need for reproducible, portable development environments.

⚡

One-Command Spawn

Spin up a complete dev environment with a single command. Fish shell, Helix editor, Zellij multiplexer, and your dotfiles ready to go.

den spawn
🧬

Reproducible Backends

Choose your philosophy. Nix for the flakes ecosystem, Guix for fully declarative Scheme configurations. Both reproducible. Both beautiful.

--guix --nix
🌐

Custom Domains

Attach your own domain to any environment. Automatic DNS via Cloudflare or sesame/Porkbun. Native attachment or forwarding fallback.

den domain
🔄

Runtime Flexibility

Run on Sprite/Fly or Railway today through one canonical Rust CLI, with a self-hosted runtime path defined so the workflow does not have to stay vendor-bound.

--runtime railway
📦

Smart Project Detection

den setup analyzes your repo and generates the perfect configuration. Detects package.json, pyproject.toml, Cargo.toml, and more.

den setup
🔒

Secure by Default

Tailscale SSH for zero-config secure access. No exposed ports. No VPN configuration. Just cryptographically authenticated connections.

tailscale
Initialization

Get Your Lab

Install den and connect your cloud accounts.

📥 Installation

# Install the canonical Rust CLI
❯ git clone https://github.com/diogenesoftoronto/den
❯ cd den
❯ cargo install --path crates/den-cli
# Verify install health
❯ den doctor --verify-auth

📋 Prerequisites

  • Rust toolchain with Cargo
  • bash and tar available locally
  • Sprite CLI or Railway CLI for runtime operations
  • sesame CLI (for Porkbun domains)
  • Cloudflare API token (optional)
# Verify installation
❯ den --version
den 0.1.0

❯ den doctor
Protocols

Mission Profiles

How developers use den in the field.

01

The Nomad Developer

Switch between laptops without losing your environment. Coffee shop MacBook, desktop Linux, borrowed Chromebook—all identical. Your entire dev setup lives in the cloud, ready when you are.

den spawn nomad-workstation && den connect nomad-workstation
02

The Clean Slate

Onboard a new teammate in minutes, not days. No "works on my machine." No dependency hell. Just a running environment that matches production exactly.

den deploy /path/to/repo --no-run && den connect repo-name
03

The Parallel Dimension

Run experiments without contaminating your main setup. Try that new language version. Test the beta compiler. If it breaks, destroy it and spawn anew.

den spawn experiment --guix && den connect experiment
04

The Remote Beast

Compile on a 32-core cloud VM from your 12" laptop. Train ML models while your laptop stays cool. Heavy lifting happens elsewhere; you stay mobile.

den exec big-project -- cargo build --release
